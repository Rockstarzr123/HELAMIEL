# HELAMIEL-JDBC

## Descripcion

Proyecto Java orientado a objetos para la gestion de productos usando JDBC y MySQL. La estructura sigue el patron MVC, separando la conexion, el modelo, el acceso a datos y la vista de consola.

Este proyecto forma parte de la creacion y seguimiento de producto virtual para la empresa HELAMIEL.

## Tecnologias

- Java 17
- Maven
- JDBC
- MySQL
- MySQL Connector/J
- Arquitectura MVC

## Requisitos

- JDK 17 instalado
- Maven instalado
- MySQL Server instalado y en ejecucion
- Base de datos creada con el script SQL del proyecto

## Instrucciones de instalacion

1. Clonar o descargar el proyecto.
2. Abrir el proyecto `HELAMIEL-JDBC` en el editor de preferencia.
3. Ejecutar el script SQL ubicado en `src/main/resources/sql/helamiel.sql`.
4. Revisar las credenciales de conexion en `src/main/java/conexion/Conexion.java`.
5. Compilar el proyecto con Maven.
6. Ejecutar la clase principal `vista.Main`.
